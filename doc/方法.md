# Method

## 1. Problem Formulation

近年来，大规模扩散模型在文本到图像生成任务中取得了显著进展。然而，由于扩散模型生成内容与训练数据之间缺乏可追溯关系，生成图像的版权归属与来源可信性问题逐渐受到关注。数字水印技术被认为是解决这一问题的重要手段。

传统图像水印方法通常在生成图像完成之后对像素进行修改，即：

[
x' = x + \Delta
]

其中 (x) 为原始图像，(\Delta) 为水印扰动。然而，这类方法存在两个主要问题：

1. 水印与图像结构耦合较弱，容易被图像后处理操作移除；
2. 嵌入过程独立于生成模型，无法利用生成模型的结构特性。

为解决上述问题，本文提出一种 **扩散过程水印方法**。与传统方法不同，我们将水印嵌入操作直接置于扩散模型的 latent 生成轨迹中，使水印信号能够在生成网络传播过程中逐渐融入图像结构。

设扩散模型的生成过程为：

[
z_T \sim \mathcal{N}(0,I)
]

[
z_{t-1} = f_\theta(z_t,t,p)
]

[
x = D(z_0)
]

其中：

* (p) 为文本 prompt；
* (z_t) 为时间步 (t) 的 latent 表示；
* (f_\theta) 为扩散模型去噪网络；
* (D) 为 VAE 解码器。

本文的目标是在扩散生成过程中嵌入水印信号，使生成图像满足：

1. **视觉质量保持稳定**；
2. **水印可通过统计检测识别**；
3. **在常见攻击下保持稳定性**。

为此，本文提出一种 **Content–Geometry Evidence Chain Watermarking Framework（CEG-WM）**，通过内容证据链与几何结构证据链实现鲁棒水印检测。

---

# 2. Overview of the Proposed Framework

CEG-WM 框架由三个核心模块组成：

1. **内容证据链（Content Evidence Chain）**
2. **几何结构证据链（Geometry Evidence Chain）**
3. **统计融合判决模块（Statistical Fusion Decision）**

整体流程如下：

[
\text{Prompt}
\rightarrow
\text{Preview Generation}
\rightarrow
\text{Content-aware Planning}
\rightarrow
\text{Diffusion-time Watermark Embedding}
\rightarrow
\text{Dual Evidence Extraction}
\rightarrow
\text{Statistical Detection}
]

方法的关键思想包括：

* 利用 **Preview Generation** 获取图像语义结构先验；
* 基于语义显著性进行 **内容自适应子空间规划**；
* 在扩散生成过程中进行 **latent 子空间嵌入**；
* 在检测阶段融合 **内容证据与几何证据** 完成统计判决。

这种设计使得水印信号能够与生成结构深度耦合，从而提升鲁棒性。

---

# 3. Preview-conditioned Content Planning

扩散模型在生成之前无法获得最终图像语义结构，因此本文首先生成一个 preview 图像，以估计语义结构并规划嵌入子空间。

## 3.1 Preview Generation

给定文本 prompt (p)，扩散模型首先生成一个预览图像：

[
x_{preview} = G_\theta(p,s)
]

其中：

* (G_\theta) 为扩散生成模型；
* (s) 为随机种子。

Preview Generation 使用与正式生成相同的条件参数，以保证两次生成的语义结构保持一致。

---

## 3.2 Semantic Saliency Estimation

为了获取图像语义结构，本文利用显著性估计函数：

[
M = \Phi(x_{preview})
]

其中：

[
M \in {0,1}^{H \times W}
]

语义掩码 (M) 描述图像中视觉显著区域。

具体而言：

* (M_{i,j}=1) 表示语义显著区域；
* (M_{i,j}=0) 表示非显著区域。

显著区域通常对应主体物体或纹理复杂区域，而非显著区域通常对应背景区域。

---

## 3.3 Content-aware Region Partition

根据语义掩码 (M)，图像空间被划分为两个互补区域：

[
R_{HF} = {(i,j) \mid M_{i,j}=1}
]

[
R_{LF} = {(i,j) \mid M_{i,j}=0}
]

其中：

* (R_{HF}) 表示高频嵌入区域；
* (R_{LF}) 表示低频嵌入区域。

这种划分使得水印嵌入能够根据图像内容结构进行自适应调整。

---

# 4. Diffusion Latent Subspace Modeling

为了在 latent 空间中寻找稳定的嵌入方向，本文对扩散轨迹特征与 Jacobian-Vector Product（JVP）估计进行联合 SVD，获得低维子空间基底。

具体地，采样 $T$ 步去噪轨迹构造中心化特征矩阵 $F_{\text{traj}} \in \mathbb{R}^{T \times d}$，并估算对应的 JVP 矩阵 $F_{\text{JVP}} \in \mathbb{R}^{T \times d}$（支持真实 JVP 或基于有限差分的代理估计），构造联合分解矩阵：

[
F = \begin{bmatrix} F_{\text{traj}} \\ F_{\text{JVP}} \end{bmatrix} \in \mathbb{R}^{2T \times d}
]

对 $F$ 进行奇异值分解：

[
F = U\Sigma V^T
]

取前 $k_{\text{LF}}$ 个右奇异向量构成 LF 子空间基底，取次 $k_{\text{HF}}$ 个右奇异向量构成 HF 子空间基底：

低频稳定子空间

[
B_{LF} \in \mathbb{R}^{d\times k_{LF}}
]

高频敏感子空间

[
B_{HF} \in \mathbb{R}^{d\times k_{HF}}
]

其中：

* LF 子空间由奇异值最大的方向确定，对应最稳定的生成方向；
* HF 子空间由次级奇异向量确定，对应图像细节敏感方向；
* 两个子空间均受语义掩码 $M$ 驱动的内容条件化调整（mask conditioning）。

这些子空间为后续水印嵌入提供结构基础。

---

# 5. Diffusion-time Dual-channel Watermark Embedding

在获得子空间规划之后，本文在扩散生成过程中对 latent 表示施加受控扰动。

扩散更新为：

[
z_t' = z_t + \Delta_t^{LF} + \Delta_t^{HF}
]

其中：

* (\Delta_t^{LF}) 为低频编码嵌入；
* (\Delta_t^{HF}) 为高频截断嵌入。

---

## 5.1 Low-frequency Encoding Channel

低频通道用于嵌入编码信息，采用伪高斯采样策略。

首先将 latent 投影到 LF 子空间：

[
c_{LF} = B_{LF}^T z_t
]

随后通过稀疏 LDPC 码对消息比特编码，生成水印码字：

[
w \in \{-1,+1\}^{k_{LF}}
]

为增强检测鲁棒性，对码字施加伪高斯采样（Pseudogaussian Sampling）：

[
\tilde{w}_i = w_i \cdot |\varepsilon_i|, \quad \varepsilon_i \sim \mathcal{N}(0,1)
]

嵌入扰动为：

[
\Delta_t^{LF} = B_{LF} \tilde{w}
]

**检测侧**通过 $\mathrm{erf}$ 函数恢复各维度后验概率：

[
p_i = \mathrm{erf}\!\left(\frac{z_i}{\sqrt{2\sigma(1+\sigma)}}\right)
]

其中 $z_i$ 为检测 latent 在 LF 子空间的投影分量，$\sigma$ 为方差参数（默认 1.5）。聚合各维度后验概率得到检测分数 $s_{LF}$。

低频嵌入具有以下优点：

1. 嵌入方向稳定；
2. 对生成质量影响较小；
3. 支持多比特信息编码（LDPC 纠错）；
4. 伪高斯采样提升检测置信度。

---

## 5.2 High-frequency Truncation Channel

在语义显著区域中，本文采用截断式嵌入策略增强鲁棒性。

首先计算高频投影：

[
c_{HF} = B_{HF}^T z_t
]

随后执行阈值截断：

[
\tilde{c}*{HF,i} =
\begin{cases}
c*{HF,i}, & |c_{HF,i}| \ge \tau \
0, & \text{otherwise}
\end{cases}
]

对应扰动为：

[
\Delta_t^{HF} = B_{HF} \tilde{c}_{HF}
]

这种截断机制能够有效抑制噪声干扰，并提升检测稳定性。

---

# 6. Geometry Evidence Chain

为了提高对几何攻击（旋转、缩放、裁剪等）的鲁棒性，本文进一步引入几何证据链。

---

## 6.1 Attention-based Anchor Extraction

扩散模型中的 Transformer token 空间提供了稳定的空间关系表示。由于真实 Self-Attention 图在当前运行时以代理方式接入，本文以 **latent 特征相关性矩阵**作为注意力图的代理估计（proxy）。

具体地，对 latent 特征矩阵按列归一化后计算 token 间余弦相关性，提取代理关系锚点集合：

[
A = \{a_1,a_2,\ldots,a_n\}
]

这些锚点反映 token 间的相对空间关系，作为几何同步的基础结构证据。

---

## 6.2 Geometric Synchronization

在检测阶段，通过估计相似变换：

[
T^* = \arg\min_T \sum_i |Ta_i - a'_i|^2
]

其中：

* (a_i) 为嵌入阶段锚点；
* (a'_i) 为检测图像中的对应锚点。

该变换可以恢复嵌入区域的空间对齐关系。

---

# 7. Statistical Detection

检测阶段从待检测图像中提取三类证据：

* LF 证据
* HF 证据
* 几何证据

分别得到分数：

[
s_{LF}, s_{HF}, s_{geo}
]

本文采用**两阶段融合判决**机制：

**第一阶段（内容主链 NP 判决）：**

内容分数由 LF 主导、HF 增强合成：

[
\Lambda_{\text{content}}(x) = s_{LF} + \beta s_{HF}
]

其中 $\beta = 0.3$，HF 为增强项；当 LF 证据不可用时内容分数为 None，判决中止。

根据 Neyman–Pearson 判决规则：

[
\Lambda_{\text{content}}(x) \geq \tau
]

则初步判定图像包含水印。阈值 $\tau$ 在校准阶段通过 NP 准则从独立 null 分布估计确定，使 $\mathrm{FPR} \leq \alpha$。

**第二阶段（几何 Rescue Band）：**

当 NP 主链判决为负且内容分数处于 rescue band 区间时，若几何证据可信（sync 状态为 ok、anchor 非代理或已同步通过），则几何门控条件触发单侧救回：

[
\text{is\_watermarked} = \mathrm{True}, \quad \text{if geo-gate passes}
]

这种非对称设计保证几何辅助的**单向救回**语义：几何证据仅用于补救内容主链的边界误判，不允许将正判决翻转为负。

---

# 8. Complexity Analysis

嵌入阶段的主要计算开销包括：

1. Preview Generation
2. 子空间计算
3. 扩散嵌入扰动

整体复杂度仍与扩散生成过程同阶。

检测阶段主要开销来自：

* 特征提取
* 统计判决

因此检测复杂度较低。

---

# 9. Discussion

本文方法具有以下优势：

1. **语义自适应嵌入**
   利用语义显著性驱动嵌入区域规划。

2. **扩散过程嵌入**
   水印信号直接嵌入扩散 latent 轨迹。

3. **双证据链检测**
   内容证据与几何证据互补。

4. **统计可验证性**
   基于 Neyman–Pearson 判决保证检测可靠性。
