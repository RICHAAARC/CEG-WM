#### 语义有效性问题

当前 embed 输入为 512×512 线性颜色梯度合成图。InSPyReNet 对合成梯度图的"显著性"不等价于真实语义场景中的显著性，"语义自适应"声明仅在真实语义图像上成立。正确修复方向为 **Preview Generation**（$x_{\text{preview}} = G_\theta(p, s)$）：embed 前以相同条件先执行一次无注入 SD3 推理，将生成图作为 SemanticMaskProvider 输入，使语义结构先验一致（参见 `doc/方法.md`）。

#### 当前阻断点与下一个阻断点

- **当前阻断（主链不自足）**：`paper_full_cuda.yaml` 无 `embed.input_image_path`，`main/` 独立运行时 SemanticMaskProvider 以 `status="failed"` 返回，injection_mode 降级为 `latent_direct_fallback`。
- **若修复后，下一个阻断点**：
  1. 语义模型权重在目标运行环境的可用性（runtime 判定项）；
  2. 真实语义图像上掩码能否稳定提取并产出非 null `plan_digest`；
  3. 真实语义图像上 LF/HF 注入信号能否被有效检出（与链路 B 相关）。

#### 链路 A 结论

> **产物字段表面可达**（`mask_digest`、`plan_digest` 非 null，`injection_mode = "subspace_projection"`，InSPyReNet model_v2 真实执行，无 fallback），**但主链不自足**。可达性依赖 `scripts/` 注入的合成梯度图；`configs/ + main/` 独立运行路径存在完全阻断。语义有效性存疑，修复方向为 Preview Generation。

```
detect_record.content_evidence_payload.score            = 78.83428775
差值 = 78.83428775 - 78.83428575 = +2e-6

→ 若使用 canonical NP 阈值（higher 规则，score > threshold）：
  78.83428775 > 78.83428575 → is_watermarked = True（极小正裕量）

结论：即使使用正确阈值，判决裕量仅 2e-6，属于准统计边界行为，
      任意微小扰动（浮点精度、不同图像、不同攻击）均可能反转判决。
      当前水印信号与阈值边界的分离度严重不足。
```

#### 当前阻断点与下一个阻断点

- **第一阻断**：当前判决实际使用 `fallback_target_fpr_test_only` 阈值（0.01），而非 canonical NP 阈值（78.834）；`scripts/` 通过 CLI 覆写 `allow_threshold_fallback_for_tests=true` 激活了 fallback。
- **第二阻断**：去掉 fallback 后，`fusion/decision.py:85-88` 要求 detect 前必须注入 `__thresholds_artifact__`，否则直接 fail-fast；单图 detect 的 NP 阈值注入路径不自足。
- **若两处均修复后，下一个阻断点**：当前 content_score 与 canonical NP 阈值之差仅 2e-6，判决稳定性上严重不足；HF L2 分数分布需大幅改善以提供足够分离裕量。

#### 链路 C 结论

> **`detect_runtime_mode = "real"`、`decision_status = "decided"` 字段可达，`execution_report` 全链为 ok。** 但判决实际使用 `fallback_target_fpr_test_only` 阈值（0.01），非 canonical NP 阈值（78.834）；单图 detect 的 NP 阈值注入路径不自足。反事实分析表明即使使用正确阈值，判决裕量仅 2e-6（极脆弱），不符合研究目标要求的 NP 判决规范。

#### FPR = 0.02 数值分析

```
混淆矩阵：tp=100, fp=2, tn=98, fn=0

TPR = 100 / (100 + 0) = 1.0 ✓
FPR = 2   / (2 + 98)  = 0.02
NP 承诺：FPR ≤ 0.01 → 实际 0.02 → 未达成（超标 2 倍）

NP 违约根因：
  校准：100 个 null 样本 → 99th 百分位数（quantile_rule="higher"）= 78.83428575
       校准意图：null 样本中最多 1/100 超过阈值（FPR ≤ 0.01）
  评估：100 个负样本（与校准 null 样本来自不同 seed/prompt 的生成图）
       2/100 负样本 content_score > 78.83428575 → FPR = 0.02

  可能根因：
  (a) 校准集与评估负样本集使用了不同 seed/prompt，评估集存在 2 个评分略高于
      校准 99th 分位数的样本（N=100 时分位精度仅 ±1 个样本）
  (b) HF L2 分数分布较窄（大部分 null 样本评分相近），校准集 99th 分位数

```

**`enable_anchor`（几何锚点消融）**：可诊断 attention anchor 是否影响几何链内部字段与同步诊断。论文化所缺条件：几何链独立贡献的正式指标、anchor 相关攻击覆盖。

1. **链路 A 的输入图像来源依赖 `scripts/` 注入**，当前 embed 输入为合成梯度图，语义自适应声明尚不成立。

3. **链路 D 的统计指标依赖脚本构造带标签 detect records**，主链统计闭环不自足，`fpr_empirical=0.02 > target_fpr=0.01`。

**最终结论**：当前产物可用于"工程上存在真实方法机制参与"的证明，但不足以支持"论文核心研究目标已通过主链真实自足到达"的结论。
