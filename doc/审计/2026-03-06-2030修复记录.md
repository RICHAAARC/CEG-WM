## 第三轮修复（2026-03-06，发布核心正负样本闭环 — 方案3 + 全局 calibrate）

### 背景

`experiment_matrix` 负样本原使用合成偏移（`base_score - 1.0`），FPR 校准统计无效；且 per-item calibrate 导致校准集与测试集重合（FPR 恒为 0）。本轮修复分两步实施：  
- **第一步**：引入基于真实 SD 推理的负样本 detect record（per-(model_id, seed) 缓存）  
- **第二步**：引入全局 calibrate 步骤，将校准集（neg_cache 全体）与测试集（per-item 攻击正样本）严格分离

### 修改文件清单

#### 1. `main/evaluation/experiment_matrix.py`（多处）

| 位置 | 改动 |
|------|------|
| `_derive_run_root()` 之后 | 新增 `_run_neg_embed_detect_for_cache(model_id, seed, config_path, batch_root, max_samples)` 函数：1) 用 canonical digest 前 16 位作缓存 key，写入 `batch_root/neg_cache/neg_{key}/`；2) 命中缓存直接返回；3) embed 阶段追加 `test_mode_identity=true`；4) detect 阶段追加 `enable_content_detect=true` + `allow_threshold_fallback_for_tests=true`；返回 `detect_record.json` 路径或 None |
| `_run_neg_embed_detect_for_cache()` 之后 | 新增 `_run_global_calibrate(batch_root, config_path, neg_detect_record_cache)` 函数：1) 过滤有效 neg 路径；2) 为每条记录追加 `label=False` / `ground_truth_source="real_neg_embed_detect"` 写入 `global_calibrate/neg_staged/`；3) 运行一次 calibrate CLI，glob 指向 `neg_staged/*.json`；4) 返回 `global_calibrate/artifacts/thresholds/thresholds_artifact.json` 路径或 None |
| `run_experiment_grid()` — 主循环之前 | 步骤(1): 按 `(model_id, seed)` 分组调用 `_run_neg_embed_detect_for_cache()`，构建 `neg_detect_record_cache`，异常时静默降级；步骤(2): 调用 `_run_global_calibrate()` 产出 `shared_thresholds_path`，失败时静默降级为 `None`；步骤(3): 遍历 grid items，将 `neg_detect_record_path` 与 `shared_thresholds_path` 注入各 item 浅拷贝 |
| `_run_stage_sequence()` | 预先读取 `shared_thresholds_path_val`（`use_shared_thresholds` 标志）和 `neg_detect_record_path_for_stage`；calibrate 阶段：`use_shared_thresholds=True` 时跳过（`continue`），跳过前预先准备 `labelled_detect_records_glob`；evaluate 阶段：优先使用 `shared_thresholds_path_val`，降级时使用 per-item 本地阈值 |
| `_prepare_labelled_detect_records_glob_for_matrix()` | 新增 `neg_detect_record_path: Optional[Path] = None` 参数；有效时从真实 neg detect record 读取 content score，写入 `ground_truth_source="real_neg_embed_detect"`；降级时回落 `base_score - 1.0` |

#### 2. `tests/test_experiment_matrix_detect_stage_input_binding.py`（1 处）

| 位置 | 改动 |
|------|------|
| `_fake_prepare_labelled_glob()` | 签名增加 `neg_detect_record_path=None` 默认参数，与新实签名保持兼容 |

### 论文级机制满足点

| 条件 | 状态 |
|------|------|
| 校准集 ≠ 测试集 | ✅ 全局 calibrate 仅用 neg_cache，evaluate 仅用被攻击正样本 |
| FPR 由独立 null 分布控制 | ✅ shared threshold 在 neg_cache 全集上估计 |
| 负样本有真实分布 | ✅ 干净图像经 identity embed + detect，非合成偏移 |
| 样本量扩展自动生效 | ✅ 增加 `experiment_matrix.seeds` 列表即可，代码无需修改 |
| 降级兼容 | ✅ global calibrate / neg 生成失败时均回落 per-item 方案 |
